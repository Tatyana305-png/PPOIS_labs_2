# Лабораторная работа №3
## Тетрис


---

## Аннотация

В данной лабораторной работе была разработана классическая игра «Тетрис» на языке Python с использованием библиотеки Pygame. Проект демонстрирует применение событийно-ориентированного программирования, объектно-ориентированного подхода, работу с конфигурационными файлами в формате JSON, реализацию звуковых эффектов и анимаций.

---

## Постановка задачи

### Цель работы

Изучить событийно-ориентированное программирование с использованием библиотеки Pygame на языке Python путем разработки полноценной игровой графической программы.

### Задачи

1. Разработать игровое графическое приложение «Тетрис»
2. Реализовать 7 типов геометрических фигур с возможностью поворота
3. Создать систему подсчета очков и уровней сложности
4. Реализовать графическое меню с командами:
   - Начать игру
   - Таблица рекордов
   - Справка
   - Выход
5. Добавить звуковые эффекты и фоновую музыку
6. Реализовать анимации (падение фигур, очистка линий)
7. Хранить настройки в JSON файлах
8. Реализовать таблицу рекордов с сохранением имени игрока

## Выбор инструментов и технологий

### Язык программирования: Python 3.12

**Преимущества:**
- Простота синтаксиса
- Богатая экосистема библиотек
- Кроссплатформенность
- Поддержка ООП

### Библиотека: Pygame 2.6.1

**Обоснование выбора:**
- Специализированная библиотека для разработки 2D игр
- Встроенная поддержка событийно-ориентированного программирования
- Удобная работа со звуком (pygame.mixer)
- Простая отрисовка графических примитивов
- Большое сообщество и документация

### Формат хранения данных: JSON

**Обоснование:**
- Встроенная поддержка в Python (модуль json)
- Человеко-читаемый формат
- Легкий вес
- Удобное преобразование в словари и списки Python
