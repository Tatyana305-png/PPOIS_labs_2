# Лабораторная работа 2
### Цели
Изучить принципы построения графического пользовательского интерфейса. Разработать программную систему с графическим пользовательским 
интерфейсом на языке Python
### Задание
Разработать оконное приложение с одним главным окном и несколькими дочерними диалогами. Вызов диалогов осуществляется через соответствующие пункты меню. 
Команды меню должны дублироваться на панели инструментов.

### Основные классы
Класс ```MainController ``` связывает модель и представление, обрабатывает действия пользователя из главного окна: открытие диалогов добавления/поиска/удаления, сохранение и загрузку данных в XML, обновление отображения данных.

Класс ```DataController``` предоставляет методы для получения всех записей, добавления новых преподавателей, поиска по критериям и удаления. Также получает уникальные значения для выпадающих списков.

Класс ```Teacher``` - модель данных преподавателя, реализованная как dataclass. Содержит поля: факультет, кафедра, ФИО, ученое звание, ученая степень, стаж работы и идентификатор. Предоставляет методы для преобразования в словарь для БД и создания объекта из словаря.

Класс ```Database``` - класс для работы с базой данных SQLite. Отвечает за подключение к БД, инициализацию таблицы teachers, добавление записей, получение всех записей, поиск по условиям, удаление по условиям и получение уникальных значений для колонок. Содержит методы для формирования SQL-запросов с параметрами.

Класс ```MainWindow``` - главное окно приложения. Содержит меню (Файл, Правка, Вид), панель инструментов с кнопками, строку состояния и вкладки для переключения между табличным и древовидным представлением. Отображает все записи преподавателей и обновляет статусную строку

Класс ```PaginatedTable``` предоставляет таблицу с возможностью навигации по страницам: кнопки "первая", "предыдущая", "следующая", "последняя". Позволяет изменять количество записей на странице (5, 10, 20, 50, 100).

Класс ```AddDialog``` диалоговое окно для добавления нового преподавателя. Содержит форму с полями: факультет (выпадающий список), кафедра (выпадающий список), ФИО (текстовое поле), ученое звание (выпадающий список), ученая степень (выпадающий список), стаж работы (счетчик).

Класс ```SearchDialog``` - диалоговое окно для поиска преподавателей с тремя вкладками: по фамилии и кафедре, по ученому званию и факультету, по стажу работы. Содержит выпадающие списки с уникальными значениями из БД. Результаты поиска отображаются в постраничной таблице внутри этого же диалога.

Класс ```DeleteDialog``` - диалоговое окно для удаления преподавателей по условиям. При выполнении удаления показывает сообщение о количестве удаленных записей или об отсутствии записей, соответствующих условиям.

Класс ```XMLWriter``` создает XML-документ с корневым элементом <teachers>, внутри которого для каждого преподавателя создается элемент <teacher> с атрибутом id и вложенными элементами для каждого поля. Форматирует вывод с отступами для читаемости.

Класс  ```XMLReader``` предназначен для чтения данных из XML-файла. Содержит обработчик TeacherHandler, который последовательно разбирает XML-файл и создает объекты Teacher при обнаружении элементов <teacher>.

```Constants``` - файл с константами приложения. Содержит списки предопределенных значений: ученые звания, ученые степени, факультеты, кафедры, а также заголовки для таблицы.

### Запуск

Приложение запускается командой:
```
python main.py

```
